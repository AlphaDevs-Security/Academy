מה כל יום מכיל עכשיו

  • וידאו פתיחה — placeholder + תסריט מילה-במילה (לאור מקליט בעצמו)
  • חימום (5 דק׳) — תרגיל מודעות עם textarea להקלדה
  • שיעורים (50-60 דק׳) — התוכן הקיים, נשאר ללא שינוי
  • תרגיל מעשי (30-40 דק׳) — פרומפט מובנה לביצוע ב-AI שלהם, צ׳ק-ליסט
  • מקרה בוחן (20-25 דק׳) — סיפור אמיתי עם timeline ותוצאות
  • חוברת עבודה (15-20 דק׳) — קישור להורדה
  • אתגר בונוס (30-35 דק׳) — אופציונלי, למתקדמים
  • רפלקציה (10 דק׳) — 4 שאלות, נשמרות ב-localStorage
  • מבחן (קיים) — נשאר כמו שהיה

הגנות נגד שיתוף לא מורשה

  • License Modal — בכניסה הראשונה דורש שם, מייל, ומפתח רישוי. נשמר ב-localStorage.
  • Watermark אישי — שם הקונה והמייל מוצגים בכל עמוד (פינה + אלכסון מקבילי)
  • Anti-share — חסימת קליק ימני, F12, Ctrl+S/U/P. הצגת אזהרה ב-console.
  • בכל מפתח בפורמט AD-XXXXXX — תקבל גישה. בייצור — ה-LMS (Thinkific) ייקח אחריות.
